# CPR Cognition Experiment

这是一个面向成人 CPR 急救引导研究的网页版原型。  
它不是商业急救产品，而是为了稳定生成可分析的实验数据，支持解释深度、时间压力、认知负荷、信任校准、纠错接受和中断恢复等变量。

## 项目目标

- 作为成人 CPR 急救引导的研究工具
- 支持中英双语切换，默认中文
- 稳定记录全过程日志
- 支持实验员控制、回放、标注和导出
- 支持后续论文分析所需的数据结构

## 已实现模块

- 参与者流程页
- 正式任务页
- 问卷页
- 实验员控制台
- 回放与标注
- JSON / CSV 导出
- 草稿保存与恢复
- 手机端适配
- Safari 兼容优化
- 中英双语切换

## 核心研究变量

### 1. 解释深度

- 对照组
- 纯指令型
- 简释型
- 全释型

### 2. 时间压力

- 低压
- 高压
- 隐藏倒计时

### 3. 场景版本

- 基础 CPR
- 打断版 CPR
- 纠错版 CPR
- 高压力 CPR

### 4. 可记录指标

- 首次响应时间
- 每一步完成时间
- 流程完整度
- 错误率
- 中断次数
- 纠错接受率
- 解释请求次数
- 重复解释次数
- 信任评分
- 认知负荷评分

## 运行方式

在项目目录中运行：

```powershell
node server.mjs
```

终端会输出本地地址和局域网地址。  
默认在浏览器中打开 `http://127.0.0.1:3000`。

## 如果要在 iPhone 上测试

- 确保电脑和 iPhone 连接同一个 Wi-Fi
- 在 iPhone 的 Safari 中打开终端输出的局域网地址
- 如果局域网无法访问，优先检查防火墙和网络隔离

## 建议实验流程

1. 说明与同意
2. 设备检查
3. 训练示例
4. 正式任务
5. 任务中引导与纠错
6. 任务后问卷
7. 回顾访谈
8. 导出 JSON / CSV

## 文件说明

- `index.html`：页面结构
- `app.js`：实验逻辑
- `styles.css`：样式
- `server.mjs`：本地服务与会话保存
- `manifest.json`：PWA 配置

- `实验员操作手册.md`：简要操作说明

## 研究写作提示

这套原型更适合在论文里表述为：

> 本研究构建了一个面向成人 CPR 的网页实验原型，用于操控解释深度与时间压力，并记录参与者在任务执行中的认知负荷、信任校准和行为表现。

这样更容易进入 HCI / CHI 的研究语境。

## 数据导出

每次实验后可以导出：

- 会话 JSON
- 行为 CSV
- 回放 JSON

这些数据适合用于：

- 量表分析
- 过程分析
- 事件编码
- 回放标注
- 条件比较

## 备注

这个系统是为了研究，不是医疗替代方案。  
真实急救情境中，请优先拨打当地急救电话并遵循当地急救规范。

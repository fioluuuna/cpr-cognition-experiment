Research tool for adult CPR guidance

CPR Cognition Experiment

A mobile-first research prototype for studying explanation depth, time pressure, cognitive load, trust calibration, interruption recovery, and action execution.

Online Condition Pressure Trial Step

Session setup

Set the participant ID, trial index, condition, pressure, and scenario before starting.

Research logic

  • Independent variables: explanation depth, time pressure, scenario variant.
  • Mechanisms: cognitive load, trust calibration, interruption recovery.
  • Outcomes: response delay, order completion, correction acceptance, execution quality.
  • Logging is session-level, trial-level, and step-level.
Supported outputs

JSON, CSV, replay index, annotations, questionnaire data.

Platform

Runs in Safari on iPhone and any mobile browser.

Participant flow

Consent, device check, training, task entry, and post-task reflection are arranged as one guided flow.

1. Study introduction

Explain that this is a CPR cognition study, not a medical product.

2. Device check

Confirm the browser, viewport, and basic interaction readiness.

3. Training / example

Show one example explanation so the participant understands how the interface behaves.

4. Start task

Enter the CPR scenario and begin the timed decision sequence.

5. Interview entry

After the task, use the interview section for retrospective comments.
